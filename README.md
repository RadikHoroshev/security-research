# Security Research

Independent vulnerability research on AI/ML infrastructure.

## Structure

```
security-research/
├── results/           — PoC scripts and vulnerability reports
├── targets/           — target research notes
├── scripts/           — analysis and scanning tools
├── scout/             — continuous monitoring
├── knowledge_base/    — vulnerability patterns and references
└── README.md
```

## Focus Areas

- AI/ML inference servers and frameworks
- Common vulnerability classes: RCE, SSRF, path traversal, authentication bypass
- Targets: LiteLLM, Ollama, Open WebUI, LangFlow, MiniRAG, Xinference, and others
